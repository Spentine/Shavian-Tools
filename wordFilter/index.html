Shavian Word Filter

How to Use

  To use the program, first select the desired filters then press the filter button. If there are words that match the filter, another menu should appear prompting to sort and output. Click the button with your desired sort and then output your desired text format. It should appear under the "Output" header.

Filters

  • The "Allowed Characters" input box will filter for words that only contain those letters.
  • The "Must Contain" input box will filter for words that contain that particular string of characters.
  • The "Starts With" input box will filter for words that start with that particular string of characters.
  • The "Ends With" input box will filter for words that end with that particular string of characters.

Sorts

  • The "Shavian" sort will sort in Shavian alphabetical order.
  • The "Latin" sort will sort in Latin alphabetical order.
  • The "Shavian Length" sort will sort the words by its length with the Shavian alphabet.
  • The "Latin Length" sort will sort the words by its length with the Latin alphabet.

Outputs

  • The JSON and CSV output is self-explanatory; they output their respective formats.
  • The "Text (Shavian & Latin)" output will display one line for each word; and on each line, the format "[Shavian] [Latin]" will be used.
  • The "Text (Shavian)" output will display all the Shavian words in order, separated by spaces.
